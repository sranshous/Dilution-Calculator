This is program to help calculate the dilutions for Cait in lab.

Input:
    M0: Stock molarity
    M1: Desired molarity
    V1: Desired volume

Output:
    V0: The required volume of stock

---------------------

Running the program

1. Download this file
2. Open the terminal and navigate to the directory where you put the file
3. Type into the terminal:
    javac DilutionCalc.java
4. To run it type into the terminal
    java DilutionCalc
